# Python Interpreter

## Overview
This is a Python interpreter that can execute Python code dynamically. It supports various features that allow users to write and test their Python scripts in a flexible environment.

## Features
- Dynamic code execution
- Support for Python 3.x
- Error handling and debugging tools
- Built-in documentation for members

## Installation
To install the Python interpreter, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/mere4567/python-interpreter.git
   ```
2. Navigate to the project directory:
   ```bash
   cd python-interpreter
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To use the interpreter:
1. Run the following command:
   ```bash
   python interpreter.py
   ```
2. Enter your Python code when prompted.

## Examples
### Example 1: Basic Calculation
```python
print(2 + 2)
```

### Example 2: Defining a Function
```python
def add(a, b):
    return a + b
print(add(5, 3))
```
